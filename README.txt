TITOLO
Easybook

Descrizione
La piattaforma permette principalmente la lettura e il download di ebooks che siano a pagamento o gratis.
La piatatforma memorizza le informazioni in un database le cui entità sono:
- ebooks,
in cui vengono memorizzati: titolo, autore, genere, data di pubblicazione, prezzo(facoltativo), descrizione, 
copertina e file in pdf.
- iscritti,
in cui troviamo: nome, cognome, email/username, password (appositamente criptata)
- acquisti,
con: titolo, copertina e pdf.
Se l'utente ha già acquistato l'ebook può leggerlo o scaricarlo dall'apposita sezione 'I tuoi acquisti'.
Inoltre all'utente è permessa tale lettura esclusicamente se registrato o il libro è gratis.
La piattaforma permette all'utente di registrarsi e di memorizzare le sue informazioni all'interno del database.
Tutti gli ebooks sono in formato pdf.
E' possibile effetturare una ricerca per titolo, genere, o autore nella barra di ricerca.
Se un utente è già registrato il sistema impedisce una nuova registrazione.
All'utente registrato viene inoltre data la possibilità di caricare un nuovo ebook nel sistema e metterlo a disposizione di altri utenti;
il catalogo viene automaticamente aggiornata con i nuovi dati inseriti.
